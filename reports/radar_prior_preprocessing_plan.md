# 雷达先验进入时空图建模的预处理基准与优先级说明

## 1. 当前研究目标

本阶段不再以“寻找最优数据预处理方法”为主线，而是基于潇杭前期实验报告中已经验证过的结论，将有效的雷达经验知识迁移到当前 ST-GNN 框架中，重点研究：

1. 雷达先验如何进入空间图建模；
2. 雷达先验如何进入时间建模；
3. 雷达先验如何作为模块输入参与融合。

因此，后续实验应避免重复大规模预处理消融，而应把前期有效方法整理为可复用模块，并明确区分“数值尺度预处理”“雷达先验特征”“模型结构改进”三类内容。

## 2. IPIX 与 SDRDSP 的核心区别

当前工作需要同时参考两类数据：一类是当前代码中正在使用的 IPIX Dartmouth 数据，另一类是潇杭前期实验中主要使用的 SDRDSP 数据。二者的数据形态、预处理状态和适合迁移的结论并不完全相同，必须先区分清楚。

### 2.1 数据来源不同

IPIX 是实测海杂波雷达数据，当前代码从 `.cdf` 文件中读取 ADC 数据，并提取 `hh/hv/vv/vh` 四种极化下的复数 I/Q。

SDRDSP 前期实验使用的是 `.mat` 格式的海上目标/海杂波数据，实验中通常通过在背景杂波上注入模拟目标，构造不同 SCR 条件下的训练和测试样本。

因此：

- IPIX 更强调真实数据、多极化、少距离单元标签；
- SDRDSP 前期实验更强调可控 SCR 注入、低 SCR 检测曲线和预处理消融。

### 2.2 距离单元规模不同

当前 IPIX 处理后的样本为：

```text
E: [num_windows, P=4, N=14]
```

前期 SDRDSP 复现实验主要使用：

```text
E: [num_samples, P=4, N=256]
```

这一区别非常关键。很多在 SDRDSP 上设计的局部参考窗方法，例如 Local RMS 中的 `G=2, K=16`，依赖较大的距离单元数，在当前 `N=14` 的 IPIX 数据上不能直接照搬。

因此，SDRDSP 上关于 PL/RPH、RD、相位差、cell-wise stats 的结论更容易迁移；而依赖大范围距离参考窗的 Local RMS/CFAR 式归一化需要重新设计。

### 2.3 标签来源不同

当前 IPIX 使用数据集文件中的目标距离单元标注，并在预处理时保存：

```text
y_range
range_labels
range_roles
primary_range_bin
secondary_range_bins
```

其中 `target_policy=related` 时，primary 与 secondary 相关距离单元都可能作为目标相关单元参与训练。

SDRDSP 前期复现实验中，目标通常是人工注入的，标签按“目标注入单元=1，其余=0”构造。严格论文复现口径强调只标注目标注入单元，而不是把目标相邻单元都标为正。

因此，比较 IPIX 与 SDRDSP 结果时不能只看 Pd/Pfa 数值，还要注意标签定义和目标比例是否一致。

### 2.4 当前预处理状态不同

当前 IPIX 预处理已经做了：

- I/Q 均值方差标准化；
- I/Q 正交不平衡校正；
- 四极化提取；
- `P=4` 滑窗；
- 距离单元级标签保存。

SDRDSP 严格按论文“无预处理”输入时，前期报告发现原始幅度过大，会导致 GAT 注意力饱和、loss 停滞、检测失效。因此 SDRDSP 前期最终采用 P99 percentile scaling 作为数值尺度修正。

这说明：

- P99 是 SDRDSP 大幅值原始数据上的必要数值修正；
- 当前 IPIX 已有标准化和校正，不应机械重复 SDRDSP 的 P99/P95 消融；
- 当前 IPIX 应以现有 `Raw I/Q` 作为基准，再逐步加入雷达先验输入。

### 2.5 可迁移结论与不可直接迁移结论

可以优先迁移的结论：

- PL/RPH 是有效的时间/运动雷达先验；
- Multi 5-channel 中的幅度、RD、相位差通道有可解释物理意义；
- cell-wise stats 有独立判别力；
- FAR 阈值应来自训练杂波分布；
- 评价需要同时看 Pd、Pfa、H0 Pfa 和 seed 稳定性。

需要谨慎迁移的结论：

- SDRDSP 上的 P99 scaling 不一定需要在当前 IPIX 上重复消融；
- Local RMS/CFAR 式局部归一化不能直接套用 `G=2, K=16`；
- SDRDSP 的 `N=256` 实验结果不能直接等同于 IPIX 的 `N=14`；
- SDRDSP 中人工注入目标的 SCR 曲线不能直接等同于 IPIX 真实标注目标检测结果。

总结来说，当前工作应当把 SDRDSP 前期实验视为“雷达先验候选库”，而不是把其中每一种预处理都原样复制到 IPIX。真正适合当前主线的是：从 IPIX 当前 Raw I/Q baseline 出发，优先迁移 PL/RPH、cell-wise stats 和先验图结构。

## 3. 当前基准实验采用的预处理

### 3.1 IPIX 当前代码基准

当前 `E:\marine_target_stgnn` 中的 IPIX 预处理已经完成以下工作：

- 从 IPIX Dartmouth `.cdf` 文件读取原始 ADC 数据；
- 根据极化方式提取 `hh/hv/vv/vh` 复数 I/Q；
- 对 I/Q 做均值、标准差归一化；
- 做 I/Q 正交不平衡校正；
- 按 `P=4` 脉冲窗口切分样本；
- 保存逐距离单元标签 `y_range`；
- 保存距离单元元信息，例如 `range_m`、`primary_range_bin`、`secondary_range_bins`、`range_roles`。

当前模型基准输入为：

```text
E: [B, P, N] complex
real/imag stack -> [B, 2, P, N]
```

当前 IPIX 数据中：

```text
P = 4
N = 14
input channels = 2: I, Q
label = y_range, 距离单元级二分类
```

该版本可以作为后续所有雷达先验注入实验的 baseline。

### 3.2 SDRDSP 前期实验基准

潇杭前期 SDRDSP 复现实验中，严格按论文“无预处理”输入原始 I/Q 时，模型出现 GAT 注意力饱和、loss 停滞、检测无效的问题。因此前期实验最终采用：

```text
Raw I/Q + P99 percentile scaling
```

具体做法是：

```text
P99 = percentile(|train clutter amplitude|, 99)
I' = real(X) / P99
Q' = imag(X) / P99
```

该结论适用于 SDRDSP 原始大幅值数据。对于当前 IPIX 数据，因预处理脚本已经做过 I/Q 标准化与校正，不应机械重复 SDRDSP 的 P99 消融，而应先把当前 IPIX 版本作为基准。

## 4. 与论文原始预处理的区别

论文原始设定更接近：

```text
原始 complex I/Q -> real/imag 两通道 -> ST-GNN
```

论文没有明确加入：

- P99 scaling；
- Local RMS 归一化；
- PL/RPH 人工特征；
- RD/FFT 多通道；
- cell-wise statistics；
- global statistics；
- U-Net 前置增强；
- 统计特征 late fusion。

前期报告中的很多方法不是论文原始方法，而是为了复现稳定性和低 SCR 检测能力所做的扩展。因此在写作和实验命名时应区分：

- “论文复现 baseline”：原始 I/Q 输入；
- “数值尺度修正 baseline”：I/Q + P99 或当前 IPIX 标准化；
- “雷达先验增强方法”：PL/RPH、RD、相位差、cell-wise stats、CFAR 参考窗图等。

## 5. 不同预处理/先验方法的优先级排序

### P0：当前必须保留的基准

#### 1. 当前 IPIX 预处理后的 Raw I/Q

优先级：最高，作为所有实验对照。

输入：

```text
[I, Q]
```

原因：

- 当前代码已经支持；
- 与论文 ST-GNN 输入形式最接近；
- 可作为后续所有先验模块的公平 baseline；
- 不引入额外人工雷达特征，便于比较先验贡献。

注意：

- 当前 IPIX 为 `N=14`，不是 SDRDSP 的 `N=256`；
- 因此不能直接套用前期 Local RMS 的大窗口参数；
- 若增加特征，必须保持训练/测试统计量不泄漏。

### P1：最优先复用的雷达先验

#### 2. PL/RPH 特征通道

优先级：最高。

输入形式：

```text
[I, Q, PL, RPH]
```

其中：

- PL：phase linearity，相位线性度；
- RPH：relative peak height，多普勒谱相对峰值高度。

可复用结论：

- 前期五方法对比中，PL/RPH 综合表现最好；
- AUC 最高；
- FAR 控制稳定；
- 参数增加很少；
- 低 SCR 目标的 `o(0)` 分布明显改善。

为什么属于雷达先验：

- PL 来自目标运动导致的稳定 Doppler 相位规律；
- RPH 来自目标能量在慢时间 Doppler 域聚集的经验；
- 二者都是雷达目标检测中的物理/统计判别线索。

适合当前方向：

- 可作为时间建模先验；
- 可作为节点特征输入；
- 也可用于构造空间图边权，例如相位/多普勒相似图。

### P1：建议同步考虑的雷达统计先验

#### 3. Cell-wise statistics

优先级：高。

输入形式：

```text
[I, Q] + cell-wise stats branch
```

典型统计量：

- local RMS；
- local P95；
- local mean；
- local std；
- contrast；
- temporal mean；
- temporal std；
- phase stability。

可复用结论：

- 前期多模态实验中，`Raw + Cell stats` 有明显增益；
- `Stats only` 也能达到较强排序能力，说明统计特征有独立判别力；
- 打乱 cell stats 后性能下降，说明模型确实使用了该信息。

为什么属于雷达先验：

- 它显式描述每个距离单元的局部幅度、相位稳定性、脉冲间起伏；
- 对应传统检测中“背景杂波统计”和“目标相干性”的经验；
- 比普通归一化更接近可解释检测依据。

注意：

- `global stats` 贡献较弱，不能作为主创新；
- 当前 IPIX `N=14`，cell-wise 特征应优先沿脉冲维计算，而不是依赖大范围距离参考窗。

### P2：可作为融合增强的先验输入

#### 4. Multi 5-channel

优先级：中高。

输入形式：

```text
[I, Q, |X|, |FFT_p(X)|, delta_phase]
```

可复用结论：

- 前期实验中稳定性最好；
- AUC 标准差低；
- 低 SCR 目标尾部压低效果明显；
- 适合与 PL/RPH 融合。

为什么属于雷达先验：

- `|X|` 对应回波幅度/RCS 线索；
- `|FFT_p(X)|` 对应慢时间 Doppler 能量分布；
- `delta_phase` 对应脉冲间相位变化和目标运动。

注意：

- 它比 PL/RPH 更像“多通道物理变换输入”；
- 输入通道增加后，应控制参数量，避免性能提升被误解释为模型容量增加；
- 建议作为 PL/RPH 后的第二阶段融合实验。

### P2：可以保留但不作为主线

#### 5. Global statistics

优先级：中低。

可复用结论：

- 前期报告中 global stats 单独使用几乎没有检测能力；
- global-only AUC 接近随机；
- global shuffle 对整体影响较小；
- 但与 raw/cell stats 组合时可能提供样本尺度信息。

为什么它不应作为主雷达先验：

- global stats 是样本级尺度信息，不包含明确的距离单元级目标位置判别；
- 容易被误解为 SCR 标签捷径；
- 对空间图建模贡献较弱。

建议：

- 可以作为辅助输入；
- 不要单独作为论文主要创新点；
- 若使用，必须设置 global-only 阴性对照，证明没有走样本级捷径。

### P3：谨慎使用

#### 6. Local Robust RMS / CFAR式局部归一化

优先级：低，暂不建议作为当前主线。

可复用结论：

- 前期实验显示低 SCR 检测潜力很大；
- 但 FAR 出现过严重崩溃；
- 原因是局部参考窗杂波过弱时，归一化尺度趋近 0，导致特征爆炸。

为什么属于雷达先验：

- 它借鉴 CFAR 的参考窗、保护窗、局部背景估计思想；
- 是典型雷达检测经验。

为什么当前不能直接复用：

- 当前 IPIX 只有 `N=14` 个距离单元；
- 前期 SDRDSP 使用的 `G=2, K=16` 不适用于小 N；
- 若强行套用，参考窗不足，估计不稳定；
- 容易破坏 FAR。

如果后续使用，必须加入：

- sigma 下限；
- 小 N 专用参考窗；
- H0 纯杂波 Pfa 验证；
- FAR 崩溃检查。

### P3：暂不优先

#### 7. U-Net 可学习预处理

优先级：低。

可复用结论：

- 前期 50 epoch 下收敛较慢；
- 长训练可能有潜力；
- 但解释性弱，和“雷达先验进入时空图建模”的主题不够直接。

为什么不优先：

- 它更像通用神经网络增强模块；
- 不如 PL/RPH、cell stats、CFAR 图结构具有明确雷达知识；
- 容易偏离老师要求的“雷达先验经验”。

## 6. 什么是可复用结论

以下结论可以直接复用，不需要重复做大规模消融：

1. 原始 SDRDSP 大幅值数据直接输入 ST-GNN 会导致训练不稳定，P99 scaling 是有效的数值尺度修正。
2. PL/RPH 是前期综合最优的显式雷达先验输入，兼具性能、稳定性和可解释性。
3. Cell-wise stats 有独立判别力，不只是附加噪声或装饰特征。
4. Multi 5-channel 稳定性较好，适合与 PL/RPH 进一步融合。
5. Global stats 单独没有距离单元级检测能力，不适合作为主创新。
6. Local RMS 低 SCR 潜力大，但存在 FAR 崩溃风险，不能直接部署。
7. FAR 阈值必须来自训练集杂波分布，不能使用测试集信息。
8. 评价不能只看 Pd，还必须同时看实际 Pfa、H0 Pfa、低 SCR 目标分布和 seed 稳定性。

## 7. 什么是雷达先验知识

可以称为雷达先验的内容包括：

### 6.1 空间结构先验

- 距离单元相邻关系；
- CUT 与左右参考单元关系；
- 保护单元与参考窗设计；
- 目标可能影响相邻距离单元的扩展关系；
- 基于距离、幅度、相位、多普勒相似度构造图边。

### 6.2 时间/运动先验

- 目标在慢时间维具有较稳定 Doppler 相位；
- 相位随脉冲近似线性变化；
- 目标 Doppler 频谱能量更集中；
- 海杂波相位更随机、谱能量更分散；
- 脉冲间相干性可作为目标判别线索。

### 6.3 统计检测先验

- 局部背景杂波功率估计；
- CFAR 参考窗/保护窗思想；
- 局部幅度统计；
- 相位稳定性统计；
- 目标与杂波在低 SCR 下的分布差异；
- FAR 阈值由训练杂波分布确定。

### 6.4 多极化/观测通道先验

当前 IPIX 有 `hh/hv/vv/vh` 极化通道，可作为潜在先验：

- 不同极化下目标和海杂波响应不同；
- 共极化与交叉极化可能提供互补信息；
- 可研究极化融合图或极化注意力。

但这一点前期 SDRDSP 实验不一定覆盖，需要在 IPIX 上单独验证。

## 8. 什么不应称为雷达先验

以下内容更适合称为数值处理、训练技巧或通用模型改进，不应单独包装成雷达先验：

- Adam/AdamW 优化器；
- scheduler；
- batch size 调整；
- 梯度裁剪；
- 类别权重；
- 普通 dropout；
- 普通 MLP/Conv 层堆叠；
- 单纯增加通道数但没有物理意义；
- 只用 global stats 推断样本整体 SCR；
- 使用测试集统计量做归一化或阈值；
- 为了提高指标而改变标签定义。

其中 P99 scaling 介于两者之间：它是必要的数值尺度修正，但本身不是强雷达先验。它不能作为“雷达经验知识进入模型”的核心创新，只能作为稳定输入尺度的 baseline 处理。

## 9. 后续实验建议

### 实验组 A：空间图先验

目标：回答“雷达先验如何进入空间图”。

建议对比：

1. 固定相邻图 baseline；
2. CFAR参考窗图；
3. 幅度相似图；
4. PL/RPH相似图；
5. 可学习图 + 先验图融合。

### 实验组 B：时间雷达先验

目标：回答“雷达运动经验如何进入时间建模”。

建议对比：

1. Raw I/Q；
2. Raw I/Q + PL/RPH；
3. Raw I/Q + RD/FFT；
4. Raw I/Q + delta phase；
5. PL/RPH + RD + delta phase 融合。

### 实验组 C：统计先验融合

目标：回答“传统检测统计量如何作为模块输入”。

建议对比：

1. Raw I/Q；
2. Raw I/Q + cell-wise stats；
3. Raw I/Q + PL/RPH；
4. Raw I/Q + PL/RPH + cell-wise stats；
5. Raw I/Q + PL/RPH + cell-wise stats + prior graph。

## 10. 当前阶段推荐路线

建议当前论文/代码主线为：

```text
Baseline:
  当前 IPIX 标准化 Raw I/Q + 原始 ST-GNN

First improvement:
  Raw I/Q + PL/RPH

Second improvement:
  Raw I/Q + PL/RPH + cell-wise stats

Third improvement:
  在空间图中引入 CFAR/相似性先验
```

也就是说，本阶段的重点不是继续问“还要不要做数据预处理”，而是：

```text
把前期已验证有效的预处理经验，转化为模型内部的雷达先验输入、图结构和时间建模机制。
```

这更符合老师任务中“提取可复用结论，避免重复实验”的要求。
