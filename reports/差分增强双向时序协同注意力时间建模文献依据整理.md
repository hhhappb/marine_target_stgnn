# 差分增强双向时序协同注意力时间建模文献依据整理

本文整理“差分增强双向时序协同注意力”作为 ST-GNN 时间特征提取器（TFE）替换模块的参考来源。该方向的目标不是改动外部输入契约，而是在论文主干

```text
FT/输入特征 -> SFE1 -> TFE1 -> SFE2 -> TFE2 -> DetectionHead
```

中的 `TFE1/TFE2` 位置做受控替换，保持：

```text
E_complex: [B, P, N]
logits:    [B, 2, N]
```

## 1. 文献分层

| 层级 | 论文 | 作用 |
|---|---|---|
| 直接方法来源 | VideoFusion: A Spatio-Temporal Collaborative Network for Multi-modal Video Fusion | 直接提供 differential reinforcement 和 bi-temporal co-attention 的结构灵感 |
| 方法思想辅助 | VideoFusion: Decomposed Diffusion Models for High-Quality Video Generation | 支持“共享背景 + 时变残差”的视频分解思想，可映射到海杂波与目标扰动 |
| 轻量时序注意力旁证 | LAVIE: High-Quality Video Generation with Cascaded Latent Diffusion Models | 支持简单 temporal self-attention 即可捕获视频时序相关性，提醒 TFE 替换应轻量 |
| Co-attention 旁证 | Spatio-temporal Co-attention Fusion Network for Video Splicing Localization | 支持多帧异常定位任务中使用 spatio-temporal co-attention 融合特征 |
| 雷达物理依据 | Small-floating Target Detection in Sea Clutter via Visual Feature Classifying in the Time-Doppler Spectra | 支持小目标会破坏海面运动连续性，差分/局部变化有检测意义 |
| 雷达建模依据 | Complex VAE with Heavy-Tailed Likelihood for Radar Target Detection in Sea Clutter | 支持直接建模 complex slow-time I/Q、重尾海杂波、CFAR 阈值，而不是转成普通图像问题 |

## 2. 六篇论文整理

### 2.1 直接方法来源：VideoFusion 多模态视频融合

**论文**：VideoFusion: A Spatio-Temporal Collaborative Network for Multi-modal Video Fusion  
**链接**：https://arxiv.org/abs/2503.23359  
**年份/状态**：2025 arXiv，2026 修订版，arXiv 页面标注 Accepted to CVPR 2026。  

**核心内容**：

- 构建红外-可见光视频融合数据集 M3SVD。
- 提出多模态视频融合网络 VideoFusion。
- 使用 differential reinforcement 做跨模态差异信息增强。
- 使用 complete modality-guided fusion 自适应融合多模态特征。
- 使用 bi-temporal co-attention 动态聚合 forward/backward temporal contexts，增强跨帧表示。

**对本项目的启发**：

这篇是“差分增强双向时序协同注意力”的直接参考源。虽然它处理的是多模态视频融合，不是雷达检测，但结构思想可以迁移：

```text
视频帧序列      -> 雷达慢时间 pulse 序列
跨帧不一致      -> 慢时间目标扰动
forward context -> 当前 pulse 之前的慢时间上下文
backward context-> 当前 pulse 之后的慢时间上下文
差分增强        -> 幅度差分、相位差分、特征差分
```

**可借用部分**：

- differential reinforcement：用于强化 `H_t - H_{t-1}`、`H_{t+1} - H_t` 或相位差分。
- bi-temporal co-attention：用于融合前向和后向慢时间上下文。
- gated fusion：用于控制差分分支、前向分支、后向分支对最终 TFE 输出的贡献。

**不能直接照搬部分**：

- 原论文的跨模态设定是红外-可见光。当前 IPIX 主线是单极化分别训练，不能把四极化当成已有外部输入契约直接混进 TFE。
- 视频融合输出是重建/融合视频；我们输出是逐 range cell 二分类 logits，因此模块必须服务于 Pfa/PD 检测。

### 2.2 方法思想辅助：VideoFusion 分解扩散

**论文**：VideoFusion: Decomposed Diffusion Models for High-Quality Video Generation  
**链接**：https://arxiv.org/abs/2303.08320  
**年份/状态**：CVPR 2023。  

**核心内容**：

- 指出现有视频扩散方法若对每帧使用独立噪声，会忽略视频中的内容冗余和时间相关性。
- 将逐帧噪声分解为所有帧共享的 base noise 和沿时间变化的 residual noise。
- 用两个联合学习的网络分别匹配共享部分和时变残差。

**对本项目的启发**：

这篇不是双向协同注意力来源，但它提供一个很适合雷达的建模比喻：

```text
视频共享内容 / base noise      -> 海杂波背景中的共享或缓变成分
视频时变 residual noise        -> 小目标引起的慢时间扰动
分解式建模                     -> 背景稳定成分与目标差异成分分开编码
```

**可应用方式**：

在 TFE 中显式引入差分分支：

```text
D_t = [H_t - H_{t-1}, H_{t+1} - H_t]
```

或引入慢时间均值/低频背景估计：

```text
B_t = temporal_smooth(H)
R_t = H_t - B_t
```

其中 `R_t` 更接近目标扰动或非平稳异常，作为 attention bias 或 gate 输入。

### 2.3 轻量时序注意力旁证：LaVie

**论文**：LAVIE: High-Quality Video Generation with Cascaded Latent Diffusion Models  
**链接**：https://arxiv.org/abs/2309.15103  
**年份/状态**：2023 arXiv。  

**核心内容**：

- 使用级联 latent diffusion 做高质量文本到视频生成。
- 论文摘要强调 simple temporal self-attentions 加 rotary positional encoding 足以捕获视频时序相关性。
- 也强调图像-视频联合微调对生成质量重要。

**对本项目的启发**：

这篇支持一个保守设计原则：TFE 替换不应一开始就做大而复杂的 Transformer。当前 ST-GNN 的优势是低 observation time 和实时性，新的时间模块应优先使用轻量注意力。

**可应用方式**：

- 对 `P` 维做小规模 temporal attention，而不是对 `[P, N]` 全局 token 做大 attention。
- attention 只在每个 range cell 内沿 pulse 维做，复杂度约为：

```text
O(B * N * P^2 * d)
```

对 IPIX `N=14`、`P=4/16` 可控；对 SDRDSP `N=256` 也比全局 `[P*N]^2` 注意力安全。

### 2.4 Co-attention 旁证：SCFNet 视频拼接定位

**论文**：Spatio-temporal Co-attention Fusion Network for Video Splicing Localization  
**链接**：https://arxiv.org/abs/2309.09482  
**年份/状态**：2023 arXiv。  

**核心内容**：

- 面向视频拼接篡改定位。
- 使用三流 encoder 捕获多帧 manipulation traces。
- 使用 parallel co-attention 和 cross co-attention 融合时空取证特征。
- 目标是定位视频中的异常或篡改区域。

**对本项目的启发**：

它说明 co-attention 不只适合生成/融合，也适合多帧异常定位。雷达小目标检测同样是从强背景中定位异常目标，只是异常的物理来源不同。

**可应用方式**：

在 TFE 中把前向上下文、后向上下文、差分上下文看作三个分支：

```text
F_forward  = context(H_1 ... H_t)
F_backward = context(H_t ... H_P)
F_diff     = context(delta H)
```

然后用 co-attention 或 cross-gating 进行协同融合：

```text
Z_t = gate_f * F_forward + gate_b * F_backward + gate_d * F_diff
```

### 2.5 雷达物理依据：Time-Doppler 小漂浮目标检测

**论文**：Small-floating Target Detection in Sea Clutter via Visual Feature Classifying in the Time-Doppler Spectra  
**链接**：https://arxiv.org/abs/2009.04185  
**年份/状态**：2020 arXiv。  

**核心内容**：

- 面向海杂波中的小漂浮目标检测。
- 论文观察到目标回波会破坏海面底层运动在 time-Doppler spectra 中的连续性。
- 使用 LBP 描述 TDS 图像纹理变化。
- 用 one-class SVM 检测相对纯杂波的偏离。
- 在 IPIX 实测雷达数据上评估。

**对本项目的启发**：

这篇给“差分增强”的雷达物理合理性提供直接支撑：

```text
海杂波：慢时间/多普勒结构具有一定连续性
小目标：破坏这种连续性，形成局部纹理、相位或频率扰动
差分增强：强化这种连续性破坏或目标扰动
```

**可应用方式**：

在 TFE 中加入雷达含义明确的差分：

```text
幅度差分: |E_t| - |E_{t-1}|
相位差分: angle(E_t * conj(E_{t-1}))
特征差分: H_t - H_{t-1}
Doppler proxy: short-window FFT magnitude / peakness / phase linearity
```

若先从 feature-space 差分做起，应在说明中写清：它是对 SFE 后慢时间扰动的可学习近似，不等同于显式 Doppler 谱。

### 2.6 雷达建模依据：Complex VAE 重尾海杂波检测

**论文**：Complex VAE with Heavy-Tailed Likelihood for Radar Target Detection in Sea Clutter  
**链接**：https://arxiv.org/abs/2606.10540  
**年份/状态**：2026 arXiv。  

**核心内容**：

- 面向海杂波重尾、尖峰多、目标标注稀缺的问题。
- 使用 complex-valued VAE 对 clutter-only complex baseband slow-time sequence 建模。
- 输入表示为 I/Q 分量，学习联合重建。
- 使用 Student-t 负对数似然刻画重尾重建误差，降低对离群点敏感性。
- 加入 time-domain amplitude error constraint。
- 推理时用重建偏差作为检测统计量，并用纯杂波验证集经验分位数设定 CFAR 阈值。

**对本项目的启发**：

这篇支持三个关键原则：

1. 雷达时间建模应尊重 complex slow-time I/Q，而不是只做普通视频图像类比。
2. 海杂波具有 heavy-tailed、spike-prone 特性，差分增强需要鲁棒门控，避免把海尖峰误当目标。
3. 最终检测仍要回到 CFAR/Pfa 约束，不能只报告分类准确率。

**可应用方式**：

- 差分分支中优先考虑复数相位差和幅度差，而不是只做普通实值 feature residual。
- 使用鲁棒归一化、门控或 clipping 抑制海杂波尖峰放大。
- 结果评估必须报告 `threshold_source`、`target_pfa`、`actual_pf`、`PD`，与当前项目规则保持一致。

## 3. 方法合理性链条

该 TFE 替换思路可以形成如下论证链：

```text
VideoFusion 2025/2026:
  差分增强 + 双向时序协同注意力可以缓解视频序列中的时序不一致和干扰。

VideoFusion 2023:
  视频序列可分解为共享背景成分和时变残差成分。

LaVie / SCFNet:
  轻量 temporal attention 与 co-attention 能有效建模多帧时空关系，不必默认使用大模型。

Time-Doppler 雷达检测:
  小目标会破坏海杂波慢时间/多普勒连续性，差分和局部变化具有检测物理意义。

Complex VAE 雷达检测:
  complex slow-time I/Q、重尾海杂波、CFAR 阈值是雷达检测必须保留的统计口径。
```

因此，本项目中的合理表述应是：

> 本文借鉴视频融合中的差分增强与双向时序协同注意力思想，将其改写为面向雷达慢时间序列的 TFE 替换模块。差分分支用于强化目标引起的幅度、相位或特征扰动；双向时序协同注意力用于在有限 observation window 内聚合前后慢时间上下文；雷达侧通过复数 I/Q、相位差、幅度差、Doppler proxy 和 CFAR/Pfa 评估约束保证该模块不是普通视频模块的直接迁移，而是服务于海杂波小目标检测的可解释时间建模。

## 4. 建议的 TFE 模块形态

候选配置名：

```text
tfe_replacement_diff_bicam
```

模块输入输出：

```text
输入: [B, C_in, P, N]
输出: [B, C_out, ceil(P / 2), N]
```

推荐流程：

```text
H
-> 差分增强分支
   D_prev = H_t - H_{t-1}
   D_next = H_{t+1} - H_t
-> 前向时序分支
-> 后向时序分支
-> 双向 co-attention / cross-gating
-> 轻量融合
-> stride=2 时间压缩
```

如果需要显式雷达先验，可在输入特征或 TFE 内部加入：

```text
amplitude difference
wrapped phase difference
short-window Doppler peakness
phase linearity
temporal residual from local mean / low-frequency trend
```

## 5. 必须做的消融

为证明该模块有效，而不是靠参数量或观测时间取胜，至少需要：

| 实验 | 目的 |
|---|---|
| original_tfe | 原论文 TFE 对照 |
| diff_only | 只验证差分增强是否有效 |
| bicam_no_diff | 验证双向协同注意力本身是否有效 |
| forward_only | 排除只靠前向上下文即可解释收益 |
| backward_only | 排除非对称上下文偶然收益 |
| no_coattention | 验证 co-attention/cross-gating 是否必要 |
| shuffle_diff | negative control，验证差分时序对应关系是否真实有效 |
| same_P_original_tfe | 若使用 P=16/P=32，必须与同 P 原 TFE 比较 |

## 6. 风险与边界

- `P=4` 时双向注意力上下文很短，收益可能有限；`P=16` 更可能体现优势，但必须区分 observation time 收益和模块收益。
- 双向时序注意力使用完整窗口，不是严格因果在线模型；实时性报告必须计入 observation time。
- 差分会放大海杂波尖峰，应加入鲁棒归一化或门控。
- 当前 IPIX 主线是四极化分别训练，不能未经实验协议修改就把多模态视频融合里的“跨模态”直接解释成四极化融合。
- 正式结论必须使用训练集或校准集杂波阈值，不能用测试集诊断阈值作为论文性能。
