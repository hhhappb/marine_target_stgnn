# 雷达先验动态图空间特征提取模块替换实验报告

生成日期：2026-06-22

## 1. 总体结论

本阶段实验表明，早期动态图模块效果差的主要原因不是“动态图思想无效”，而是替换位置和动态图建模方式没有严格对齐论文 ST-GNN 的 SFE 模块语义。将动态图重新设计为 ST-GNN 主干中的 SFE1/SFE2 直接替换模块后，`P=16` 下 `radar_prior_dynamic_sfe` 在当前快筛口径中取得最好结果：

```text
Original SFE P16:           PD=0.022642, PF=0.001049
Radar-prior dynamic SFE P16: PD=0.050943, PF=0.001049
Delta PD:                  +0.028302
```

进一步消融说明，性能提升不是单纯来自静态距离图，也不是单纯来自动态相似图，而是来自二者的组合：

```text
静态距离先验提供物理约束，避免动态图被海杂波相似性误导；
局部动态相似图提供目标延展与局部 range cell 相关性；
少量 top-k 非局部动态补边提供额外远程散射关联。
```

## 3. 起点：错误替换与早期动态图效果不佳

最初的动态图实验包括 `dynamic_spatial_graph`、`adaptive_dynamic_spatial_graph` 等早期模块。这些模块的问题主要有两类：

1. 研究口径问题：早期实现容易被理解成“自定义空间处理模块”，没有充分强调它必须严格替换论文 ST-GNN 的 SFE1/SFE2。
2. 图建模问题：动态关系容易退化为过强的全局混合，远距离杂波 cell 也参与传播，和我们预想的“局部为主、少量远程相关补边”的动态图不同。

早期 P=4/16/32 快筛结果如下，日志来源：

```text

```

| adaptive_dynamic_spatial_graph | 4 | 0.004245 | 0.001049 | +0.000943 |

- P=16/P=32 的 original SFE 本身由于更长观测时间获得提升。
- 当时的动态图模块没有真正吃到更长慢时间窗口带来的好处。

## 4. 修正路线：严格替换 ST-GNN 的 SFE blocks

后续修正后，正式主线不再使用旧的自定义骨架，而是使用：

```text
paper_modules/models/sfe_replacement_stgnn.py
```

该骨架保持论文 ST-GNN 顺序：

```text
FT/输入特征 -> SFE1 -> TFE1 -> SFE2 -> TFE2 -> DetectionHead
```

并将空间模块替换位置限定为：

```text
SFE1/SFE2
```

新动态图模块为：

```text
paper_modules/models/modules/spatial_graphs/radar_prior_dynamic_sfe.py
```

核心设计如下：

1. 静态距离先验图

```text
A_static = gamma / (|i-j| + 1) + (1 - gamma) * 1(|i-j| <= delta)
```

含义：

- 距离越远，空间相关性越弱；
- 局部窗口内的 range cells 额外增强；
- 对应 HRRPGraphNet++ 中“距离衰减 + 局部结构”的物理归纳偏置。

2. 动态相似图

对每个 range cell 提取当前窗口内的慢时间签名：

```text
signature_i = [P, C] flatten
```

再计算 cell 之间的归一化相似性：

```text
score_ij = cosine(signature_i, signature_j) / temperature
```

含义：

- 图结构不再固定；
- 相似性来自当前观测窗口；
- 动态图反映当前样本中 range cell 的局部目标延展或散射相关性。

3. 受约束动态边

动态图不是全连接 attention，而是：

```text
local window edges + top-k nonlocal similar edges
```

这一步是关键修正。它避免动态图把所有远距离杂波 cell 混在一起，也更符合我们最初预想的动态图：

```text
局部邻接是主体；
少量远距离相关 cell 作为补充；
远程关系必须由当前样本相似性证明。
```

4. 静态图与动态图融合

```text
A = static_weight * A_static + (1 - static_weight) * A_dynamic
```

当前主线配置：

```text
static_gamma = 0.5
static_delta = 5
static_weight = 0.7
dynamic_topk = 2
dynamic_temperature = 0.2
dropout = 0.1
```


## 6. 贡献来源消融：静态距离图还是动态相似图？

```text

```

| 配置 | 含义 |

| no nonlocal | 静态距离先验 + 局部动态边，不加 top-k 非局部边 |

## 7. 消融结论

### 7.1 单独静态距离图不是主要贡献

`static only` 的 PD 只有：

```text
0.007783
```

低于 original SFE P16 的：

```text
0.022642
```

这说明单纯使用 HRRPGraphNet++ 风格的距离衰减图并不能解决当前 IPIX 海杂波小目标检测问题。原因可能是：

- IPIX 只有 `N=14` 个 range cells，空间范围很短；
- 海杂波与目标的空间结构并不只由距离决定；
- 固定静态图缺少对当前观测窗口的适应能力。

### 7.2 单独动态相似图也不是主要贡献

`dynamic only` 的 PD 只有：

```text
0.008255
```

也低于 original SFE P16。这说明如果完全放弃雷达距离先验，只根据当前窗口相似性连边，动态图会被海杂波相似性误导。

这和前期失败经验一致：无约束动态图容易把背景杂波传播到目标 cell，导致低 Pfa 下检测分数校准变差。

### 7.3 最大贡献来自“静态约束下的局部动态边”

`no nonlocal` 已经达到：

```text
0.044575
```

明显超过 original SFE P16：

```text
0.022642
```

这说明主贡献来自：

```text
静态距离先验 + 局部动态相似边
```

也就是：

- 图结构仍然尊重 range cell 的局部空间连续性；
- 局部边权根据当前慢时间签名动态变化；
- 相邻或近邻 cell 是否强相关，由当前样本数据决定。

### 7.4 非局部 top-k 补边提供额外收益

full dynamic SFE 比 no nonlocal 更高：

```text
0.050943 - 0.044575 = 0.006368
```

说明少量非局部相似边确实有帮助，但它不是最大贡献来源。更准确地说：

```text
局部动态边是主体贡献；
top-k 非局部边是增益项；
静态距离先验是防止动态图跑偏的约束项。
```

## 8. 试错过程中的关键认识

本阶段最重要的研究收获不是某一个 PD 数字，而是明确了动态图在本任务中应该如何建模。

错误方向：

```text
把动态图当作普通 attention；
让所有 range cells 全局互相传播；
只搬用 HRRPGraphNet++ 的静态距离图；
只用相似性动态边而不加雷达空间约束；
没有严格声明替换的是 ST-GNN 的 SFE1/SFE2。
```

正确方向：

```text
固定 ST-GNN 主干，只替换 SFE blocks；
保持 model(E_complex) -> [B, 2, N]；
静态距离先验负责物理约束；
动态相似边负责当前样本适应；
局部窗口是主体；
少量非局部 top-k 边是补充；
通过消融验证每个组成部分是否必要。
```

## 9. 可写入论文的阶段性表述

可以将本阶段方法描述为：

> We replace the original SFE blocks in ST-GNN with a radar-prior constrained dynamic spatial graph module. The proposed graph combines a static range-distance prior and a sample-adaptive similarity graph derived from slow-time signatures. The static prior prevents over-mixing among unrelated range cells, while the dynamic graph adaptively strengthens target-related local structures and introduces a small number of non-local correlated edges.

中文表述：

> 本文在 ST-GNN 的 SFE 模块位置引入雷达先验约束动态图。该图由静态距离先验图和样本自适应动态相似图融合而成。静态图通过距离衰减和局部窗口约束 range cell 间的物理关联，避免动态图被海杂波相似性误导；动态图则基于当前慢时间窗口的 range cell 签名动态调整边权，并通过少量 top-k 非局部边补充远程相关散射结构。

消融结论可表述为：

> Ablation results show that neither the static range prior alone nor the dynamic similarity graph alone improves detection performance. The performance gain mainly comes from their combination, indicating that radar-informed structural constraints are necessary for reliable dynamic graph modeling in sea-clutter target detection.

## 10. 下一步建议

后续应继续围绕 `P=16 + radar_prior_dynamic_sfe` 主线推进，但必须从快筛转向正式评估。

优先级建议：

1. 保留当前 full dynamic SFE 作为空间图主线。
2. 使用更大训练窗口和完整测试窗口重新跑 P=16。
3. 增加至少 3 个 seed，验证提升不是 seed 偶然。
4. 将阈值来源从 `test_diagnostic_current_eval` 改为训练集杂波或独立校准集杂波。
5. 继续做参数消融：
   - `static_weight = 0.5 / 0.7 / 0.9`
   - `dynamic_topk = 1 / 2 / 3`
   - `static_delta = 3 / 5 / 7`
6. 对比输入通道增强实验，将 `real_imag` 与 `real_imag + amplitude/phase/delta phase` 和当前动态图组合，验证二者是否互补。
7. 最终回到论文 Fig. 7 口径，按 14 个 IPIX 文件和四极化完整报告 PD。

## 11. 当前报告结论

本阶段已经形成一条清晰的实验链：

```text
早期动态图效果差
-> 定位到替换位置和图建模语义问题
-> 清理旧模块
-> 严格在 ST-GNN SFE1/SFE2 位置替换
-> 构造雷达先验约束动态图
-> P=16 下性能显著增强
-> 消融证明静态图或动态图单独使用均无效
-> 证明有效贡献来自静态距离约束下的局部动态相似建模，以及少量非局部补边
```

因此，当前空间动态图主线具备继续投入价值，并且已经有足够的实验过程与消融证据支撑后续论文方法章节和实验章节的撰写。
